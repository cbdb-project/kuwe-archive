## KUWE

The KUWE project is a joint initiative by the Harvard Digital China Initiative and the China Biographical Database Project. The project aims to collect the most useful large language model projects for East Asian Studies. It is currently maintained by Kwok-leong Tang, Hongsu Wang, Wenxin Xiao, and Helen He.
